#  Sistema de Gerenciamento de Contratos

Uma aplicação Full-Stack desenvolvida em React para o monitoramento inteligente de prazos contratuais e gestão de documentos administrativos.

##  Funcionalidades Principais
* **Painel de Monitoramento**: Visualização clara de todos os contratos ativos.
* **Alertas de Prazos Críticos**: Sistema visual que destaca contratos com vencimento inferior a 65 dias (Barra de alerta no rodapé e destaque nos cards).
* **Gestão de Reajustes**: Campo dedicado para controle de Reajuste e Reequilíbrio Econômico-Financeiro.
* **Integração em Nuvem**: Armazenamento de dados em tempo real e links diretos para contratos em PDF.

##  Tecnologias Utilizadas
* **React.js**: Construção de interface dinâmica e responsiva.
* **Firebase Firestore**: Banco de dados NoSQL para persistência de dados.
* **Firebase Hosting**: Hospedagem da aplicação para acesso remoto.
* **CSS3**: Estilização personalizada com foco em experiência do usuário (UX).
